# **App Name**: NexusSight

## Core Features:

- Secure User & Project Management: Establish a robust user authentication and authorization system, enabling creation and management of multiple projects with associated user roles and permissions.
- Multi-Project Dashboard: Provide a centralized overview for all projects, displaying key metrics, progress trackers, and an interactive element for selecting different projects.
- Document & Drone Video Hub: Allow users to securely upload, organize, and view project-related documents and drone visualization videos. Integrate basic media playback capabilities.
- Geospatial Map Integration: Display project locations, drone flight paths, and other geographical data on an interactive map, supporting the overlay of relevant visual information.
- AI Voice Command Interface: Leverage Gemini AI as a tool for voice command recognition, allowing users to navigate modules, access data, and receive contextual in-app guidance for forms or modules using natural language.
- Responsive & Adaptive Layout: Ensure the application design fluidly adapts to various screen sizes, providing an optimal user experience across desktops, tablets, and smartphones, including a top-level collapsible navigation menu.

## Style Guidelines:

- Primary accent color: A sophisticated, muted cerulean (#2D577D) representing professionalism and data integrity. This color will be used for primary calls to action, headings, and interactive elements, providing a strong visual anchor.
- Background color: An airy, barely-there blue-grey (#EFF3F6), offering a clean, expansive canvas that enhances readability and reduces visual fatigue for extended use. Its high brightness and low saturation contribute to the app's objective, corporate aesthetic.
- Secondary accent color: A vibrant yet composed turquoise (#26AFD9), selected for its energetic contrast while maintaining a professional appeal. This hue will highlight key information, provide immediate feedback on interactive states, and draw attention to important metrics, positioned subtly against the primary palette.
- Global font: 'Inter', a contemporary sans-serif typeface, will be utilized for all text elements—from headlines to body copy. Its objective, highly legible design aligns with the corporate ethos, ensuring clarity and an unwavering modern aesthetic across all device types.
- Icon style: A refined set of minimalistic, outline-style icons. These will convey functionality clearly and directly, complementing the objective corporate visual language without adding unnecessary visual weight or distraction. Consistent sizing and line weight are paramount.
- Layout strategy: A responsive grid system will define content placement, prioritizing optimal use of space and readability on any device. Generous use of whitespace will frame modules and data, emphasizing a clean, uncluttered interface. The main navigation will feature a top-level, collapsable menu for efficient content access.
- Interaction animations: Subtle, deliberate micro-animations will provide instant feedback on user actions (e.g., button clicks, form submissions) and smooth transitions between views. Animations will be minimal and efficient, enhancing user experience without disrupting the app's corporate and objective flow.
